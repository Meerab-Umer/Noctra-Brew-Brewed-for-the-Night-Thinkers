# ☕ Noctra Brew – Premium Coffee Brand Website

A modern, fully responsive luxury coffee brand website designed for **Noctra Brew**.  
This project showcases a premium UI/UX experience with real-world e-commerce features, built with the help of AI-powered design tools.

---

## 🌐 Live Demo
https://noctra-brew-luxury-site.lovable.app

---

## 📌 Overview
Noctra Brew is a high-end coffee brand concept inspired by night aesthetics and bold flavors.  
The website is designed to deliver a smooth, elegant, and interactive user experience while reflecting a premium café atmosphere.

---

## ✨ Features

- 🖤 Premium modern UI design  
- 📱 Fully responsive (mobile + desktop)  
- 🛒 E-commerce layout (cart, product cards)  
- 🔍 Smart search & product filtering  
- ☕ Coffee collection showcase  
- 🔁 Subscription section  
- 💬 Contact form + live chat UI  
- 🎯 Smooth animations & micro-interactions

## 🤖 AI Tools Used

This project was designed with the help of AI tools to improve efficiency and creativity:

- **Lovable AI** – used to generate website layout, UI structure, and component ideas  
- AI prompting techniques were used to define brand identity and user experience  

---

## 🧠 Prompt Overview

A structured AI prompt was used to generate this website concept. The prompt included:

- Brand identity (**Noctra Brew – premium coffee brand**)  
- Target audience (professionals & premium users)  
- Website features (shop, search, subscription, contact, etc.)  
- UI/UX style (minimal, elegant, luxury theme)  

This resulted in a more accurate and professional output instead of a generic design.
